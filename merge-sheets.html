Merge Google Sheets

Google Workspace

SUMMARY

This Google Apps Script accepts a URL to a Google Drive folder that contains several source files (Google Sheets, Excel, and CSV). The script duplicates a sheet template, merges the source files based on a common key, and populates the templated sheet. The output sheet includes a dashboard that summarizes key information in the spreadsheet.

Accept multiple source files

Users pull data from their systems of record into a Google Drive folder.

All files share a common primary key that is used to merge their data. This is either a SKU number, UUID, or website URL.

Run the merge

Users leverage an admin sheet where they enter the folder URLs of their sources and desired output location.

A custom menu item is used to start the merge process.

Merged sheet

The script merges all the source files into a single sheet.

This flexible system enables users to change the columns of the template without having to change the script.

Summary dashboard

A summary dashboard aggregates the information in the sheet.

The dashboard immediately reflects any change in the underlying data.
