Email Google Sheet as PDF

Google Apps Script

SUMMARY

A Google Sheet contains a summary sheet that aggregates data from 12 monthly sheets. A sidebar enables users to PDF and email a single monthly sheet or the entire workbook, with customizable email recipients, subject, and body.

Data aggregation

Users enter sales data in individual monthly sheets. A summary sheet dynamically aggregates the monthly data

Team members need to email individual sheets or the entire workbook as PDF attachments.

Custom data delivery

Users leverage a custom menu to open a sidebar. The sidebar lets users select between emailing a specific monthly sheet or the entire workbook.

The sidebar is prepopulated with recipients' email addresses, email subject lines, and email body text. Users can change any of these fields.

Dynamic data delivery

When the user clicks the EMAIL PDF button, a Google Apps Script gets the data from the sidebar fields. Based on the user's selection, the script creates a PDF file.

The script ensures that only monthly tabs can be sent as individual PDFs. And if users selects to send the entire workbook, then the script removes unneeded sheets, such as a config sheet and the instructions sheet.

The script sends an automated email based on the sidebar information.

PDF file

The email contains the PDF file as an attachment. For workbook PDFs, the script ensures that only populated monthly sheets will be included in the PDF. Empty monthly sheets are ignored.
