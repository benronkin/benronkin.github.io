Google sheet to PDF

Google Workspace

  • Key highlights

  • Google Apps Script that generates PDF files, such as invoices, from Google Sheet rows
  • Executes on a highlighted row in a Google Sheet
  • Duplicates a Google Doc template file
  • Populates template files with Sheet row data
  • Converts doc to PDF file

Summary

I needed an invoice generator that takes Google Sheet rows and converts them to formatted PDF files. Created A Google Apps Script that is invoked from a custom menu option in Google Sheet and converts the sheet row into a PDF by populating the row data into a formatted Google Doc template.

Tabulated invoice rows

All invoice information is included in a Google Sheet, with one row per invoice.

The Google Apps Script is launched via a custom menu while an invoice row is selected.

Invoice template

An invoice template is formatted as a Google Doc. The template contains placeholders for the invoice information.

When the script runs, it copies the template to a new file, collects the row information from the sheet, and enters each data item in its correct place inside the new document.

Formatted PDF file

Finally, the Google Apps Script converts the Google Doc to a PDF, moves the PDF to the right Google Drive, and removes the redundant Google Doc for cleanup.
